1. PRODUCT SUMMARY

The 30 Day AI SaaS Challenge is a daily, gamified app experience that guides beginners from zero to a working AI SaaS MVP in 30 days.

Each day delivers a structured micro-task that takes 5 minutes or less, powered by:

AI-generated suggestions

Pre-filled templates

Done-For-You outputs

A micro-decision

A micro-reflection

A micro-personalisation step

The app includes:

Auto-name generator

Domain availability checker

Keyword generator

Auto-legal builder

Tool setup wizard (with affiliate links)

MVP builder tools

Landing page generator

Daily reminder emails

A premium badge system

An admin dashboard for editing everything

Optional daily video lessons managed through admin panel

By Day 30 the user has:

a validated idea

a structured MVP

three generated core screens

a complete logic prompt

keywords

a name + domain

legal documentation

tool setup completed

a polished landing page

a fully summarised MVP

5. DAILY TASK STRUCTURE (MANDATORY FORMAT)

Each day uses the 4-step meaningful + frictionless model:

1. AI Output (80 percent done)

Done-For-You output

3 suggested alternatives

Shuffle button

2. Micro Decision

User picks ONE option that personalises the path.

3. Micro Reflection

Short introspective question (tap selection) to build ownership.

4. Micro Personalisation

Tiny tweak such as:

tone

colour

one word change

small feature preference

5. Completion Reward

XP

Day marked complete

Streak updated

Badge if triggered
